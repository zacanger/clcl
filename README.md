# clcl

X11 CLi CLipboard manager.

**Important**: Requires X and systemd.

**WIP**

This program is still in progress. See the roadmap section for stuff that
doesn't work yet.

--------

## Installation

`npm i -g clcl`

## Usage

```
  clcl -h      # print help
  clcl -l      # list all
  clcl -s [n]  # select entry at index n
  clcl -r [n]  # remove entry at index n
  clcl -c      # clear all
```
