# clcl

**WIP**, doesn't do anything yet.

CLi CLipboard manager.

--------

## Installation

`npm i -g clcl`

## Usage

```
  clcl -h      # print help
  clcl -l      # list all
  clcl -s [n]  # select entry at index n
  clcl -r [n]  # remove entry at index n
  clcl -c      # clear all
```
